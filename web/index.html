ζ

Zeta Playground

English-like educational programming language compiled client-side in WebAssembly

Variables

Declare with global (constant) or local (mutable), and reassign with is now:

global pi is 3.14
local radius is 5
radius is now 10

Data Types

  • INT: Whole numbers (e.g. 42, -5)
  • DECIMAL: Floats (e.g. 3.14, 0.02)
  • STRING: Written in curly braces (e.g. {Hello, World!})
  • BOOL: Logical values (true, false)

Input & Output

Print values with tell and read inputs into mutable variables with ask:

tell {Type radius:}
ask radius
tell {Area: } pi * radius ^ 2

Operators

Math precedence from highest to lowest:

  • ^ (Exponentiation, right-associative)
  • *, /, % (Multiplication, division, modulus)
  • +, - (Addition, subtraction)

Comments

Zeta supports single and multi-line comments:

< Single line comment >
<< Multi-line comment 
   continues here >>
main.zeta
Line 1, Col 1
1
